Database Setup

PrepShare Firestore Seeder

Bootstrap all courses, modules, lessons, and mock test collections in one click.

1 Enter Firebase Configuration

Go to your Firebase Console, select your project, open Project Settings (gear icon), scroll down to Web apps, and copy the config object.

2 Admin Bootstrapping Credentials

Enter the email/password to create or sign in to the platform administrator account.

Note: If Firestore rules are locked, the script will register the user as a student first. You will need to manually change their role to admin in the console before writing other collections.
